# Uber Data Analysis Dashboard
## Project Overview

This project presents an Uber Data Analysis Dashboard created using Power BI to analyze ride booking data and transportation trends.

The dashboard helps analyze trip patterns, ride frequency, booking trends, and location based insights. It converts raw Uber ride data into meaningful visualizations that support better understanding of ride sharing activity.

## Objectives

The main objectives of this project are:

• Analyze Uber ride booking patterns

• Identify peak booking periods

• Study trip distribution across locations

• Understand ride demand trends

• Provide interactive data visualization for better insights

## Dataset Description

The dataset used in this project contains Uber ride related information such as:

• Trip ID

• Pickup Date and Time

• Pickup Location

• Drop Location

• Trip Distance

• Trip Duration

• Fare Amount

• Payment Type

This dataset helps analyze transportation patterns and ride demand.

## Tools and Technologies Used

The following tools were used in this project:

• Power BI

• Data Modeling

• Data Visualization

• DAX (Data Analysis Expressions)

## Dashboard Features

The Uber dashboard includes the following visualizations:

• Total Trips Overview

• Trip Distance Analysis

• Pickup Location Distribution

• Drop Location Analysis

• Time Based Ride Trends

• Fare Analysis

These visualizations help understand ride sharing patterns and demand trends.

## Key Insights

Some insights obtained from the dashboard include:

• Identification of peak ride booking hours

• Locations with highest trip demand

• Average trip distance and duration

• Fare distribution across different trips
